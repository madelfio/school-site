Gather Source Information

To write a bibliography for a paper, first ensure you have some kind of pointer to all the sources you consulted in writing. Some popular sources of bibliographic information are:

  • ACM Digital Library
  • DBLP
  • Google Scholar

Collect bibtex entries into single file

BibTeX looks like this:

@misc{Anon06,
  author = "Nobody Anonymous",
  title = "My Article",
  year = "2006" 
}

In this example, @misc is the entry type, Anon06 is the reference key (used to reference the source from within the LaTeX document), and author, title, and year are source attributes, set to the quoted values. Depending on the entry type and the bibliography style used in a LaTeX file, different attributes will be included in the generated bibliography text.

Bibliography Entry Types

The most common entry types in my bibliographies are:

  • @article (an article published in a journal)
  • @book
  • @inproceedings (a conference paper)
  • @misc (URL, other publication)

And occasionally:

  • @techreport
  • @phdthesis
  • @mastersthesis
  • @manual

I use the crossref attribute of @inproceedings to reference:

  • @proceedings

Conference papers

Standard attributes:

@inproceedings{Same08,
  author    = {Hanan Samet and Jagan Sankaranarayanan and Houman Alborzi},
  title     = {Scalable network distance browsing in spatial databases},
  booktitle = {SIGMOD Conference},
  year      = {2008},
  pages     = {43-54},
  crossref  = {SIGMOD08},
}
@proceedings{SIGMOD08,
  booktitle = {SIGMOD '08},
  year      = {2008},
  address   = {Vancouver, Canada},
}

Notes

"Taming the Beast" is a good reference for all things BibTeX. Look there for tips on BibTeX syntax, how to deal with foreign or non-standard author names, non-ASCII characters, and other useful tricks.
