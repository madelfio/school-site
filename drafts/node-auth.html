How to add OAuth to a Node.js app using Passport

I want to have usernames and sessions in some node.js apps. Here's how I set it up using Passport (which has a user guide, but does not seem to have a cookbook-like tutorial for getting everything set up).

Install

First install Passport and the Google "Strategy" (not a fan of this name):

$ npm install passport passport-google

Files

You will be editing app.js and adding several files to your app:

./my-app
├── app.js  (Modify existing)
├── auth.js
├── public
│   └── images
│       └── google.png
└── views
    ├── login.ejs
    └── unauthorized.ejs

App.js

First, require passport, passport-google (or other strategies), and the soon-to-be-created auth.js:

var fs = require('fs'),
    express = require('express'),
    ...
    /* Authentication */
    passport = require('passport'),
    GoogleStrategy = require('passport-google'),
    auth = require('./auth');
